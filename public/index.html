App URL not configured

This APK only contains a shell. The real app is your Next.js site on the web. You must point Capacitor at that URL, then sync and rebuild the APK.

1. In the project root, add to .env or .env.local:

CAPACITOR_LIVE_URL=https://your-deployed-nextjs-domain.com

Use your real production or staging URL (must be https:// unless you enable cleartext for local HTTP).

2. Run:

npx cap sync android
npm run android:apk

Local dev (emulator): run npm run dev on your PC, then set CAPACITOR_LIVE_URL=http://10.0.2.2:3000 (Android emulator maps this to your machine’s localhost).
